Decathlon 110m haie

Les règles sont les suivantes :
1. Vous commencez par lancer les 5 dés (en cliquant le bouton Allez! ) dont la somme des valeurs sera votre score.
2. Si vous n'êtes pas satisfait du résultat, vous avez le droit de les relancez.
3. Attention! Vous n'avez que 5 chances de relancez. Seul le dernier résultat sera enregistré.
